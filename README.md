## Install

You can install mnet-ui-base using either of the methods given below.

For npm users:

```shell
  $ npm install mnet-ui/mnet-ui-base#release styled-components --save
```

For Yarn users:

```shell
  $ yarn add mnet-ui/mnet-ui-base#release styled-components
```
